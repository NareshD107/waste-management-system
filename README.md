# Centralized Waste Management System - Frontend

A premium React-based dashboard for tracking and managing organization-wide waste generation, hazardous material disposal, and sustainability targets.

## Quick Start

### 1. Prerequisite
Ensure you have [Node.js](https://nodejs.org/) installed (v18 or higher recommended).

### 2. Installation
Navigate to the `Front-end` directory and install dependencies:
```bash
npm install
```

### 3. Running the Development Server
Start the local development server:
```bash
npm run dev
```
The application will be available at `http://localhost:5173`.

## Features
- **Modern Dashboard**: Real-time stats and waste generation trends using Recharts.
- **Waste Logging**: Easy-to-use form with categories for Bio-degradable and Non-bio-degradable waste.
- **Secure Access**: JWT-based authentication context and protected routes.
- **Premium UI**: Glassmorphic design system built with vanilla CSS.

## Tech Stack
- **Framework**: React 19 + Vite (SWC)
- **Icons**: Lucide React
- **Charts**: Recharts
- **Animation**: Framer Motion
- **Routing**: React Router DOM
