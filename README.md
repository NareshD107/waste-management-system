# Centralized Waste Management System - Frontend

A premium React-based dashboard for tracking and managing organization-wide waste generation, hazardous material disposal, and sustainability targets.

## Quick Start

### 1. Prerequisite
Ensure you have [Node.js](https://nodejs.org/) installed (v18 or higher recommended).

### 2. Installation
Navigate to the `Front-end` directory and install dependencies:
```bash
npm install
```

### 3. Running the Development Server
Start the local development server:
```bash
npm run dev
```
The application will be available at `http://localhost:5173`.
